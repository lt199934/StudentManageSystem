## 安装

### 下载文件

```
git clone https://gitee.com/lt199934/stu.git
```

### 安装数据库

打开压缩文件以后找到DB并在本地MySQL数据库运行sql文件



## 运行

在idea中成功运行以后，浏览器输入地址

```
http://localhost:8080/studentManageSystem/login.jsp
```

根据数据库用户名和密码进行登录
